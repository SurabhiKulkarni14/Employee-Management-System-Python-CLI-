# Employee-Management-System-(Python CLI)


A simple **menu-driven command line application** built using Python to manage employee records.
This project demonstrates the use of Python data structures, CRUD operations, and nested dictionaries.

---

## Features

* Implemented **CRUD operations** (Create, Read, Update, Delete) using Python lists and dictionaries.
* Designed **nested data structures** to model employee documents and skills.
* Built a **menu-driven console application** using loops and decision statements.
* Implemented **search, update, and delete functionality** for employee records.
* Stored employee details including **name, ID, salary, role, documents, and skills**.

---

## Technologies Used

* Python
* Lists
* Dictionaries
* Nested Dictionaries
* Loops
* Match-Case (Python 3.10+)

## Employee Data Structure

Each employee record contains:

* Name
* ID
* Salary
* Role
* Documents

  * Aadhaar Number
  * PAN Number
* Skills (List of skills)

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/employee-management-system.git
```

2. Navigate to the project folder

```
cd employee-management-system
```

3. Run the program

```
python employee_management.py
```
